    About China Postal Codes

    China uses a 6-digit postal code system. The first two digits represent the province or autonomous region, the third digit represents the postal district within the province, and the last three digits represent the specific delivery office.

    Our database covers cities and towns across all 26 provinces and autonomous regions in China, including major cities like Beijing, Shanghai, Guangzhou, and Shenzhen.

    How do I find a postal code in China?

    To find a postal code in China, enter a city name or address in our search box above. Select your location from the autocomplete suggestions, and the postal code will be displayed alongside an interactive map showing the exact delivery area.

    What is the format of a China postal code?

    China uses a 6-digit numeric postal code system. For example, a valid postal code in China would look like '100000'. The format consists of 6 digits, where the first two digits represent the province, the last four the delivery area.
